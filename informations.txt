---- Architecture ----
RouteEstimate / Trajets / EventCost / Chauffeur / Véhicule /

Trajets -> A vide / Prestataire
EventCost -> péage / essence

---- DataType ----
Argent → BigDecimal
Distances → long
Consommation → Double
Dates → OffsetDateTime

---- Flux création mission ----

POST /api/missions
        │
        ▼
MissionController
        │
        ▼
MissionFacadeService.createMission()
        │
        ├── 1. MissionService.create()
        │
        │       ▼
        │
        │   mission table
        │
        │
        ├── 2. Récupération :
        │
        │       DriverRepository
        │       VehicleRepository
        │
        │
        ├── 3. RouteRequestFactory
        │
        │       Mission
        │       Driver
        │       Vehicle
        │
        │              ▼
        │
        │          RouteRequest
        │
        │
        ├── 4. RouteOptimizationService
        │
        │       RouteRequest
        │              │
        │              ▼
        │          HERE API
        │              │
        │              ▼
        │        RouteResponse
        │
        │
        └── 5. MissionRouteEstimateService
                │
                ▼
        MissionRouteEstimate
                │
                ▼
        mission_route_estimate table

---- Responsabilité des classes ----

MissionController
    |
    |-- reçoit les requêtes HTTP
    |-- retourne les réponses

MissionFacadeService
    |
    |-- orchestre le scénario métier
    |-- crée mission
    |-- déclenche calcul route
    |-- sauvegarde estimation

MissionService
    |
    |-- CRUD Mission
    |-- MissionRepository

RouteRequestFactory
    |
    |-- transforme :
    |
    |   Mission
    |   Vehicle
    |   Driver
    |
    |        ↓
    |
    |   RouteRequest

RouteOptimizationService
    |
    |-- appelle HERE
    |-- parse la réponse
    |-- calcule les coûts
    |
    |        ↓
    |
    |   RouteResponse

MissionRouteEstimateService
    |
    |-- transforme :
    |
    |   Mission
    |   RouteResponse
    |
    |        ↓
    |
    |   MissionRouteEstimate
    |
    |-- sauvegarde

Repositories
    |
    |-- accès PostgreSQL
